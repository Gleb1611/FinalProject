Google created the game in 2014 for the Chrome browser. It appears when users have opened a browser without an internet connection. The goal is to pass the time. A Chrome engineer named Edward Jung noted that the T-Rex dinosaur is launched more than 27 million times every month! The main character is a pixel dinosaur. But why was it not a little man or a turtle that became the hero? Chrome designer Sebastian Gabriel explained that the game invites the user to move to a prehistoric era when no one knew about the existence of the Internet. It turned out very symbolic.
